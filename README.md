# AFJ-Mobile

## 1. 工程简介

一个 iOS 示例 Demo，展示 iOS 开发中可能用到的一些技术。

### 2.2 初始化工程

打开 Mac 终端，进入工程根目录并执行 setup.sh 脚本

推荐安装 fvm、rvm、nvm，并确保使用 Flutter(3.0.5(stable))、nodejs(14.16.0)、ruby(2.7.4 ) 版本环境执行以下脚本

```shell
sh setup.sh
```
